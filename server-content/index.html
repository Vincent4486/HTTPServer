Welcome to your local development server

Server Details

  • Root Directory: server-content
  • Config Source: config.json
  • Serving From: Executable directory
  • Default Page: index.html

Usage Instructions

  • Place your static files (HTML, CSS, JS, images) inside the server-content folder.
  • Access files via your browser using paths like http://localhost:PORT/filename.
  • Update config.json to change the content directory or other settings.
  • Restart the server after modifying configuration.
  • Separate CSS and JavaScript files allow for better organization and performance.

Debug Tips

  • Check logs in the same directory as the executable.
  • Ensure server-content exists and has proper permissions.
  • Use absolute paths in config if needed for testing.
  • Browser DevTools can help debug network requests and styles.
  • Check the JavaScript console for any runtime errors.
